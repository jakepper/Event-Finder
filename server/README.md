# Server

## Install dependencies

With yarn
```bash
yarn
```

With npm
```bash
npm install
```

## Development

### .env
Copy the contents of `.env.example` into a new file called `.env`.

### Running the server
Start the server by running:

With yarn
```bash
yarn dev
```

With npm
```bash
npm run dev
```

## Production
Build the server by running:

With yarn
```bash
yarn build
```

With npm
```bash
npm run build
```

from the `server/` directory.

Build the client-side application by running:

With yarn
```bash
yarn build
```

With npm
```bash
npm run build
```

from the `server/client/` directory.

Start the server by running:

With yarn
```bash
yarn prod
```

With npm
```bash
npm run prod
```

from the `server/` directory.

Then visit [event-finder](http://localhost:3000/event-finder/home)
